Helix Swarm Guide (2022.1)

Specs: Spec fields

Get spec information

Summary

Get spec file fields.

GET /api/v11/specs/{spec}/ fields

Description

Get fields for a spec type.

Parameter Description Type Parameter Type Required

spec

Spec type. Valid spec types and location in the Helix server spec depot:

  • branch: //spec/branch/
  • change: the change spec is not stored in the Helix server spec depot
  • client: //spec/client/
  • depot: //spec/depot/
  • group: //spec/group/
  • job: //spec/job/
  • label: //spec/label/
  • license: //spec/license.p4s
  • protect: //spec/protect.p4s
  • spec: //spec/spec/
  • stream: //spec/stream/
  • triggers: //spec/triggers.p4s
  • typemap: //spec/typemap.p4s
  • user: //spec/user/

string

path

Yes
fields

Omitting the fields parameter or including the fields parameter with no fields specified, returns all of the fields in the spec including any custom fields.

Limit the response to specific fields by including a list of comma-separated fields in the request. The fields available depend on the spec type requested. Invalid fields do not cause an error and are ignored.

 array of strings path No
